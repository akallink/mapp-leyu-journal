# mapp-leyu-journal

## 项目简介

`mapp-leyu-journal` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这些页面内容相互独立，以静态 HTML 文件的形式存储，不依赖于任何特定域名或外部服务。

本仓库适合用于存放个人或小型项目中的静态页面、临时展示页、学习笔记、实验性页面等。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 可选：仓库根目录入口页
├── pages/             # 存放各独立 HTML 页面的目录（建议）
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 公共资源目录（可选）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：推荐将每个独立 HTML 页面放置于此目录，便于管理。
- `assets/`：如多个页面共用 CSS、JS 或图片资源，可统一存放于此。

## 页面归档说明

- 每个 HTML 页面均为**独立完整**的静态文件，可直接在浏览器中打开。
- 页面之间无强依赖关系，可单独更新或删除。
- 仓库不限定页面主题、用途或风格，归档内容由维护者自行决定。

## 维护说明

- 新增页面：在 `pages/` 目录下创建新的 `.html` 文件即可。
- 更新页面：直接修改对应 HTML 文件，建议保留版本历史。
- 删除页面：移除对应文件，并确保无其他页面引用。
- 本仓库不提供自动部署或构建流程，所有页面以原始静态文件形式提供。

## 许可

本仓库内容由维护者自行决定许可协议。如未明确声明，默认保留所有权利。
